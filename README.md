Fork of the [NestJS TypeScript starter](https://github.com/nestjs/typescript-starter) that adds docker-compose for my own benefit.

## Compile and run the project

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod

# run in container
$ docker-compose up
```

## Run tests

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
